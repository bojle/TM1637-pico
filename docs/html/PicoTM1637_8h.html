void TM1637_put_2_bytes ( uint  startPos,
uint  data 
)

Display one or two bytes of raw data on the display.

Parameters
startPosThe digit index to start at. Ranges from 0 to 3, where 0 is to the left
dataThe data for one or two bytes, the least significant byte will be put to the left.

For example TM1637_put_2_bytes(2, 0x3f05) will show the number 10 on the right half of the display.
